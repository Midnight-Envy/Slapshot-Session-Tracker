# Slapshot Session Tracker

## Overview

Slapshot Session Tracker is a React application designed for players of Slapshot Rebound who want to track performance across multiple play sessions.

Users can upload match JSON files generated by the game, organize those files into sessions, select the players they care about, and view aggregated statistics and trends over time.

The goal is to transform raw match data into an easy-to-read player tracking dashboard that helps users understand their performance and the performance of their friends.

---

# User Stories

## Feature 1 - Upload and Create Sessions

**User Story:** As a user, I want to upload one or more Slapshot match JSON files so that I can create a session and save it for future analysis.

### Details

- Upload one or more JSON files
- Parse files in React
- Generate a session record
- Save session to json-server
- Update React state immediately after creation
- Display the newly created session

---

## Feature 2 - Manage Sessions and Track Players

**User Story:** As a user, I want to view my saved sessions and select which players I care about so that I can focus on myself and my friends.

### Details

- Fetch saved sessions using a GET request
- Display sessions as cards
- View players discovered within each session
- Select tracked players using checkboxes
- Save tracked player selections

---

## Feature 3: Player Analytics Dashboard

### User Story

As a user, I want to view analytics for tracked players so that I can evaluate performance across multiple sessions and identify player strengths.

### Details

- Aggregate player statistics from uploaded sessions
- Calculate cumulative performance metrics
- Display tracked players as analytics cards
- Calculate advanced metrics including:
  - Points
  - Win Rate
  - Goals Per Game
  - Assists Per Game
  - Points Per Game
- Generate rule-based scouting insights based on player performance

---
